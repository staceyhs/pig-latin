## Specs

* _The program should add "way" to the end of words that begin with a vowel_
    **Input:** _I_
    **Output:** _Iway_
* _The program should move all of the first consecutive consonants to the end and add "ay" for words that begin with a consonant._
    **Input:** _Hello_
    **Output:** _Ellohay_
* _The program should move "qu" together if it is at the start of a word._
    **Input:** _Quiet_
    **Output:** _Ietquay_
* _The program will treat "y" as a consonant if the word begins with "y"._
    **Input:** _Yes_
    **Output:** _Esyay_
